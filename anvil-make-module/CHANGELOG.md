# Changelog

All notable changes to the Anvil Make.com module will be documented in this file.

## [1.0.0] - 2026-02-23

### Added
- Initial release of the Anvil Make.com module
- **Fill a PDF Template** - Fill PDF templates with JSON data
- **List PDF Templates** - Retrieve all PDF templates in organization
- **Create an E-Signature Packet** - Create e-signature packets with signers
- **Send a Draft E-Signature Packet** - Send draft packets to signers
- **Download Documents** - Get download URLs for completed documents
- **List Workflows** - Retrieve all workflows in organization
- **Get Workflow Data** - Retrieve workflow submission data and status
- **Generate PDF from HTML** - Create PDFs from HTML/CSS content
- **Make an API Call** - Universal module for arbitrary API calls

### Features
- API Key authentication with environment selection (Production/Development)
- Comprehensive error handling with meaningful messages
- Support for all common Anvil field types
- Test mode for safe development
- Detailed parameter help text
